# Instagram Cookies Export Qilish - Qo'llanma

## üìù Mavjud Extension'lar orqali Cookies Export

### Method 1: EditThisCookie (Chrome/Edge) ‚≠ê

**Agar EditThisCookie extension o'rnatilgan bo'lsa:**

1. **Instagram.com ga kirish:**
   - Chrome/Edge'da `https://www.instagram.com` ga kiring
   - Instagram account'ingizga login qiling

2. **EditThisCookie ochish:**
   - Browser toolbar'dagi EditThisCookie icon'iga bosing
   - Yoki: `chrome://extensions/` ‚Üí EditThisCookie ‚Üí "Details" ‚Üí "Extension options"

3. **Cookies Export qilish:**
   - EditThisCookie oynasida "Export" tugmasini bosing
   - Format: **Netscape HTTP Cookie File** ni tanlang
   - "Save" tugmasini bosing
   - Fayl nomi: `instagram_cookies.txt`

4. **Serverga yuklash:**
   - Faylni serverga yuklang:
     ```bash
     /var/www/sardor/data/www/download.e-qarz.uz/storage/app/cookies/instagram_cookies.txt
     ```

### Method 2: Cookie-Editor (Firefox/Chrome)

**Agar Cookie-Editor extension o'rnatilgan bo'lsa:**

1. **Instagram.com ga kirish:**
   - Browser'da `https://www.instagram.com` ga kiring
   - Instagram account'ingizga login qiling

2. **Cookie-Editor ochish:**
   - Browser toolbar'dagi Cookie-Editor icon'iga bosing
   - Yoki: F12 (DevTools) ‚Üí Application ‚Üí Cookies ‚Üí instagram.com

3. **Cookies Export qilish:**
   - Cookie-Editor oynasida "Export" tugmasini bosing
   - Format: **Netscape** ni tanlang
   - "Save" tugmasini bosing
   - Fayl nomi: `instagram_cookies.txt`

4. **Serverga yuklash:**
   - Faylni serverga yuklang

### Method 3: Browser DevTools orqali (Extension yo'q bo'lsa)

**Chrome/Edge DevTools:**

1. **Instagram.com ga kirish:**
   - `https://www.instagram.com` ga kiring va login qiling

2. **DevTools ochish:**
   - F12 yoki Right-click ‚Üí "Inspect"
   - "Application" tab'iga o'ting

3. **Cookies ko'rish:**
   - Chap panelda: Storage ‚Üí Cookies ‚Üí `https://www.instagram.com`
   - Barcha cookies ko'rinadi

4. **Cookies Export qilish:**
   - Har bir cookie'ni qo'lda copy qiling
   - Quyidagi formatda fayl yarating:

**Netscape format misol:**
```
# Netscape HTTP Cookie File
# This file was generated by EditThisCookie
# EditThisCookie is a Chrome extension
.instagram.com	TRUE	/	FALSE	1735689600	sessionid	YOUR_SESSION_ID_HERE
.instagram.com	TRUE	/	FALSE	1735689600	csrftoken	YOUR_CSRF_TOKEN_HERE
.instagram.com	TRUE	/	FALSE	1735689600	ds_user_id	YOUR_USER_ID_HERE
```

**Format:**
```
domain	flag	path	secure	expiration	name	value
```

**Muhim cookies:**
- `sessionid` - Eng muhim!
- `csrftoken` - Muhim
- `ds_user_id` - Foydali

### Method 4: curl orqali (Advanced)

Agar browser'dan cookies export qilib bo'lmasa:

1. **Browser'dan cookies'ni copy qiling:**
   - F12 ‚Üí Application ‚Üí Cookies ‚Üí instagram.com
   - Har bir cookie'ni qo'lda copy qiling

2. **cookies.txt yarating:**
   ```bash
   nano /var/www/sardor/data/www/download.e-qarz.uz/storage/app/cookies/instagram_cookies.txt
   ```

3. **Netscape formatda yozing:**
   ```
   # Netscape HTTP Cookie File
   .instagram.com	TRUE	/	FALSE	1735689600	sessionid	YOUR_SESSION_ID
   .instagram.com	TRUE	/	FALSE	1735689600	csrftoken	YOUR_CSRF_TOKEN
   ```
